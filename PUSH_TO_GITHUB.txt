                        下一步: VPS部署
================================================================================

1. SSH连接到VPS:
   ssh your_username@your_vps_ip

2. 克隆你的仓库:
   git clone https://github.com/YOUR_USERNAME/01exchange-grid-bot.git
   cd 01exchange-grid-bot

3. 运行一键安装:
   chmod +x install_vps.sh
   ./install_vps.sh

4. 配置环境变量:
   nano .env
   # 修改 SOLANA_PRIVATE_KEY

5. 启动程序:
   source venv/bin/activate
   python main.py
